# A股实时K线选股

本地个人版选股网站：日线策略筛选，分钟 K 线做盘中确认。数据来自新浪实时行情与东方财富 K 线接口，结果仅供研究观察，不构成投资建议。

## 运行

```powershell
python app.py --host 127.0.0.1 --port 8000
```

打开：

```text
http://127.0.0.1:8000
```

## 公网部署

这个项目包含 Python 后端接口，不能只用 GitHub Pages。推荐部署方式：

1. 把代码推到 GitHub。
2. 在 Render 新建 Web Service，连接该 GitHub 仓库。
3. Render 会读取 `render.yaml` 和 `Dockerfile`，自动用 Docker 启动：

```bash
python app.py --host 0.0.0.0 --port $PORT
```

Render 部署后会给出一个公网 HTTPS 地址。公网环境访问新浪、东方财富、腾讯行情接口时，可能受第三方接口限流或地域网络影响；页面会展示数据源时间和异常提示。

## 功能

- 候选列表：按“回春法/二波回踩”评分排序。
- 全 A 主流板块扫描：股票池留空时，会自动扫描沪深主板、创业板、科创板等主流 A 股，并输出前十推荐。
- 个股详情：日 K、MA5/MA10/MA20/MA60、分钟 K 线、成交量、技术面分析、基本面快照、风险验证。
- 策略回测：用历史日线在过去某一天选前十，并计算到最新交易日的收益、最大浮盈、最大回撤。
- 数据状态：展示报价时间、分钟 K 线时间、数据源异常。
- 风险提示：显示观察价、止损价、信号失效条件。

## 数据源

- 新浪 `hq.sinajs.cn`：实时价、盘口、成交额。
- 东方财富 `clist/get`：全市场主流 A 股行情列表，用于全市场初扫。
- 东方财富 `push2his.eastmoney.com`：1/5/15/30/60 分钟 K 线。
- 腾讯 `fqkline/get`：前复权日 K，用于策略筛选和均线计算。
- 新浪 `CN_MinlineService.getMinlineData`：东方财富分钟 K 线失败时的分时线备用源。

接口会做短时间缓存和限流式使用，避免高频刷新导致临时封禁。
非交易时段会使用行情源返回的最新可用交易日数据；页面摘要会显示“最新交易日”。
